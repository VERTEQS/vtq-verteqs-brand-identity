# verteqs-brand-identity

## 概要

WEB プロジェクト

## セットアップ

```bash
# リポジトリをクローン
git clone git@github.com:nozaqix/vtq-verteqs-brand-identity.git
cd vtq-verteqs-brand-identity

# 依存関係をインストール
npm install

# 開発サーバー起動
npm run dev
```
